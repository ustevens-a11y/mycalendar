Sync your data across iPhone and Chrome using Supabase (free). Your credentials are pre-configured — just run the SQL below once.

  1. Go to supabase.com → your project → SQL Editor
  2. Click New Query, paste and run this SQL:
CREATE TABLE IF NOT EXISTS cal_sync (id bigint generated always as identity primary key, user_id text not null unique, data jsonb not null default '{}'::jsonb, updated_at timestamptz default now());

📱 Getting this on your iPhone

  1. Download mycalendar.zip from Claude
  2. Right-click → Extract All → Extract
  3. Go to app.netlify.com → your site → Deploys
  4. Scroll down to the drop zone → drag the mycalendar folder onto it
  5. Copy your Netlify URL (e.g. cozy-rose-abc.netlify.app)
  6. Open that URL in Safari on your iPhone (must be Safari)
  7. Tap the Share button → Add to Home Screen → Add

💻 Accessing on Chrome (desktop)

  1. Open Chrome on your computer
  2. Type your Netlify URL in the address bar and press Enter
  3. Bookmark it so you can get back easily

🔄 Updating the calendar

  1. Get the new index.html from Claude
  2. Go to github.com/ustevens-a11y/mycalendar
  3. Click on index.html → click the pencil icon → scroll down → Commit changes
  4. Or click Add file → Upload files → drag new file → Commit changes
  5. Wait 2 minutes and refresh your URL
